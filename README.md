# Вендинговая система - Личный кабинет Франчайзера

## Требования для запуска

- .NET 8.0 SDK (скачать: https://dotnet.microsoft.com/download/dotnet/8.0)
- SQLite (уже встроен)

## Как запустить

```cmd
cd VendingAPI
dotnet restore
dotnet run
_________________________________________________________________
Для запуска проекта:
1. Установить .NET 8.0 SDK
2. Скачать или клонировать репозиторий
3. Запустить VendingAPI → dotnet run
4. Запустить VendingApp → dotnet run
5. Войти: rushan.chernyshov@inbox.ru / HFGDRET765
```
